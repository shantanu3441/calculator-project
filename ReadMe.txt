How to Run the Calculator
Create the Project Folder: Create a new folder for your project.

Create Files: Inside the folder, create three files: index.html, styles.css, and script.js.

Copy Code: Copy the corresponding code above into each file.

Open in Browser: Open index.html in your web browser to see and use the calculator.

Functionality
Basic Operations: You can perform addition, subtraction, multiplication, and division.
Clear Function: The C button clears the display.
Error Handling: If an invalid operation is attempted, the calculator shows "Error".
Feel free to customize the styles and functionality further!
